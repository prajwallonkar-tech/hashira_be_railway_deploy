## Workflow

When working on a JIRA ticket:

1. Move ticket to **In Progress** when you start
2. Include ticket ID in commit messages: `feat(auth): Add session endpoint (HASH-42)`
3. Link PR to JIRA ticket in PR description
4. Move ticket to **In Review** when PR is open
5. Move to **Done** after merge
